## Getting Started

### Prerequisites

For run this project locally you must has installed:
* Node Package Manager (NPM)
* Git
* yarn


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bjvalmaseda-dev/pokedux.git
   ```
3. Install all dependency
   ```sh
   yarn
   ```
4. Run the project
   ```sh
   yarn start
   ```
